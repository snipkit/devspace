## DevSpace Desktop

DevSpace desktop automatically checks for updates on GitHub every 15minutes.
When an update will be available this popup will appear to confirm the update process:

<figure>
  <img src="/docs/media/devspace-update-available.png" alt="DevSpace Flow" />
  <figcaption>Create Workspace</figcaption>
</figure>

The update will continue in background. When it's done you'll be prompted to restart the application.

<figure>
  <img src="/docs/media/devspace-update-done.png" alt="DevSpace Flow" />
  <figcaption>Create Workspace</figcaption>
</figure>

At restart the application will be at latest version.

## DevSpace Desktop - alternative

If you installed DevSpace using deb or rpm packages, you'll need to [download the
new package](https://dev.khulnasoft.com/releases/latest/), and install it using your package manager again.

## Optional: Update DevSpace CLI

If you just installed the CLI, you just need to download the latest version again
to update.
